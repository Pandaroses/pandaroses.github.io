Floppa Notes

Floppa notes is a smart and minimal note taking app. it uses react for the frontend, rust for the backend, uses GitHub OAuth2 for authentication, and has MongoDB as the database for storing notes incorporates a home-built text editor. uses REST API to communicate with the backend
